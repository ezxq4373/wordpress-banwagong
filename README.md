## 一、为什么 WordPress 建站要用 VPS，而不是虚拟主机

很多人刚开始建站，会直接去买虚拟主机，理由是便宜、简单，装个 WordPress 一键搞定。但用一段时间就会发现问题：

**共享资源，稳定性差。** 虚拟主机是几十上百个网站挤在一台机器上，邻居有人跑高流量，你的网站就跟着卡。

**限制多。** PHP 版本不能随便改，插件稍微复杂点就跑不起来，更别说自定义服务器配置了。

**扩展难。** 网站访问量稍微上来，立刻就要被要求升级到更贵的套餐，升了还是虚拟主机。

VPS 就不一样了。独立资源，可以装任何想装的东西，WordPress、数据库、SSL 证书，全部自己掌控。搬瓦工的 VPS 跑 WordPress 是标准玩法，稳定性能打，国内访问速度也有保障（特别是 CN2 GIA 线路）。

## 六、手把手教程：搬瓦工 VPS 安装 WordPress

### Step 1：购买并登录搬瓦工 VPS

👉 [点击这里前往搬瓦工购买页面](https://bwh81.net/aff.php?aff=77528)

选好套餐和机房，支付宝付款完成后，进入搬瓦工后台，找到你的 VPS，记下：
- **IP 地址**
- **SSH 端口**（搬瓦工不用默认 22 端口）
- **root 密码**（第一次登录后台可以重置）

### Step 2：用 SSH 连接到 VPS

**Windows 用户：** 下载 Xshell（免费版够用），新建连接，输入 IP 和端口，用户名 root，密码填刚才记下来的。

**Mac 用户：** 打开终端，输入：
bash
ssh root@你的IP -p 你的端口号


连接成功，你就进入了 VPS 的命令行界面。

### Step 3：安装宝塔面板

宝塔面板是 Linux 服务器管理的图形化工具，装 WordPress 直接在网页界面里点就行，不用敲太多命令。

Debian/Ubuntu 系统安装命令（搬瓦工默认推荐 Debian 12）：
bash
wget -O install.sh https://download.bt.cn/install/install-ubuntu_6.0.sh && sudo bash install.sh ed8484bec


安装过程需要几分钟，完成后会显示宝塔面板的访问地址和初始账号密码，**一定要记下来**。

### Step 4：登录宝塔面板，安装建站环境

浏览器打开宝塔给出的地址（类似 `http://你的IP:端口/xxxxx`），登录后会提示选择建站环境，推荐选 **LNMP**（Nginx + MySQL + PHP）组合，点击一键安装，等待大概 10-20 分钟安装完毕。

### Step 5：添加 WordPress 站点

宝塔面板左侧菜单，点击 **网站** → **添加站点**：

- **域名：** 填你的域名（确保域名已经解析到这台 VPS 的 IP）
- **数据库：** 勾选创建，填写数据库名和密码（记下来）
- **PHP 版本：** 选 8.1 或 8.2

点击提交，站点创建完成。

### Step 6：上传并安装 WordPress

两种方式：

**方式一（推荐）：宝塔一键部署**
宝塔面板 → 软件商店 → 一键部署 → 搜索 WordPress → 选择刚才创建的站点 → 一键部署，宝塔会自动下载并配置好 WordPress。

**方式二：手动上传**
去 [wordpress.org](https://wordpress.org) 下载最新版压缩包，通过宝塔面板的文件管理器上传到站点根目录，解压后访问域名，按提示填写数据库信息完成安装。

### Step 7：配置 HTTPS（免费 SSL 证书）

网站做好后，给它加上 HTTPS 是必须的，一方面安全，另一方面对 SEO 友好。

宝塔面板 → 网站 → 点击你的站点 → SSL → 选 **Let's Encrypt** → 申请证书 → 开启强制 HTTPS。

整个过程一两分钟搞定，完全免费。

### Step 8：优化 WordPress 性能

WordPress 装完默认性能一般，做几个基本优化：

**安装缓存插件：** WP Super Cache（免费）或 WP Rocket（付费，更强）。

**安装 BBR 加速（VPS 层面）：**
bash
echo "net.core.default_qdisc=fq" >> /etc/sysctl.conf
echo "net.ipv4.tcp_congestion_control=bbr" >> /etc/sysctl.conf
sysctl -p


**开启 CDN：** 如果网站有海量图片或者访问量大，可以考虑 Cloudflare 免费 CDN，进一步提升访问速度。
